# Ticketing System

A web-based ticketing system built with Node.js, Express, MongoDB, and Handlebars.

## Features

- User Authentication (Login/Register)
- Role-based access (Admin/User)
- Create, view, and manage tickets
- Comment system
- Status tracking
- Real-time updates

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Handlebars (HBS)
- JWT Authentication
- Bootstrap CSS
